# Commerce AdPilot 사용자 시나리오

## 개요
이 문서는 Commerce AdPilot의 핵심 타깃인 마케팅 팀장을 중심으로, 실제 업무 상황에서의 사용 시나리오와 기대 성과를 정리합니다. 모든 시나리오는 MVP 범위(로그인 없는 진입, CSV 업로드, 멀티 채널 동시 생성, 편집/복사/내보내기) 안에서 작성됩니다.

---

## 대표 시나리오: 월간 신상품 운영

### 사용자 프로필
- **이름**: 박수진
- **나이**: 35세
- **직업**: 온라인 쇼핑몰 마케팅 팀 리더
- **경험**: 마케팅 경력 10년, 현재 팀의 5명이 담당할 상품 수 200개/월
- **목표**: 하나의 상품 정보로 채널별 카피를 동시에 생성해 월간 작업 시간을 70% 이상 단축

### 상황
박수진 팀은 월 200개 신상품에 대해 블로그, 인스타그램, 페이스북 카피를 제작합니다. 기존 방식은 채널별로 문안을 따로 작성해 반복 작업이 크고, 검수까지 포함하면 리드타임이 길어집니다. 팀은 초안 작성 시간을 최소화하고 캠페인 기획과 성과 점검에 집중하려고 합니다.

### 작업 흐름

1. **즉시 진입 및 작업 시작**
   - 대표 페이지 접속 후 로그인 없이 바로 생성 화면으로 이동
   - 팀의 기본 작성 기준 입력:
     - 기본 톤: "친근하면서도 프로페셔널"
     - 핵심 메시지: "고품질, 합리적 가격, 빠른 배송"
     - 금지 표현/필수 표현 간단 메모

2. **상품 정보 입력(대량 업로드)**
   - CSV 파일로 200개 상품 정보 업로드
   - 필수 항목: 상품명, 카테고리, 주요 특징, 타겟 고객
   - 핵심 상품은 개별 입력으로 추가 맥락(행사 문구, 강조 포인트) 보강

3. **멀티 채널 콘텐츠 동시 생성**
   - 생성 채널 선택: 블로그, 인스타그램, 페이스북
   - 톤앤매너 선택: 전문적/감성적/유머러스한
   - 생성 버튼 클릭 후 상품별 채널 카피를 동시 생성
   - 초기 결과는 상품당 10초 이내 확인

4. **결과 검토 및 빠른 수정**
   - 생성된 카피를 채널별 미리보기로 확인
   - 길이, 표현 방식, CTA 문구를 팀 가이드에 맞게 편집
   - 필요한 항목만 재생성해 품질 보정

5. **배포 준비 및 활용**
   - 확정 카피를 클립보드 복사 또는 텍스트 파일로 내보내기
   - 각 판매 채널 운영 도구에 붙여넣어 즉시 게시 준비
   - 입력부터 최종 복사/내보내기까지 상품군 단위 작업을 10분 내 완료

### 기대 결과
- **시간 절감**: 채널별 개별 작성 대비 월간 문안 작업 시간 70% 이상 단축
- **처리량 향상**: 월 200개 상품을 동일 인력으로 안정적으로 처리
- **일관성 확보**: 채널별 톤 차이는 유지하면서 브랜드 메시지는 일관되게 반영
- **실행 속도 개선**: 초안 작성보다 검수/개선 중심으로 업무 전환
- **운영 효율 증대**: 복사/내보내기 중심의 단순한 배포 흐름으로 실무 적용 속도 향상

## 추가 시나리오 A: 시즌 캠페인 긴급 대응

### 상황
대형 할인 시즌 시작 1일 전, 박수진 팀은 급하게 80개 프로모션 상품의 채널별 문안을 업데이트해야 합니다. 기존 방식으로는 기한 내 반영이 어렵습니다.

### 작업 흐름
1. 기존 CSV를 불러와 행사 대상 상품만 추려 재업로드
2. 톤앤매너를 "행동 촉구" 중심으로 선택하고 채널별 카피 동시 생성
3. 할인율, 기간, 재고 임박 문구를 편집 단계에서 일괄 반영
4. 최종 문안을 클립보드 복사/파일 내보내기로 운영툴에 즉시 반영

### 기대 결과
- **마감 준수**: 캠페인 시작 전 전 상품 문안 업데이트 완료
- **속도 확보**: 긴급 작업에서도 입력-내보내기 리드타임을 10분 내 수준으로 유지
- **일관성 유지**: 행사 메시지를 모든 채널에 동일하게 적용

---

## 추가 시나리오 B: 신규 카테고리 런칭

### 상황
팀이 처음 다루는 신규 카테고리(예: 리빙/키친) 상품 50개가 입점되었습니다. 기존 카피 템플릿이 없어 초안 작성 부담이 큰 상태입니다.

### 작업 흐름
1. 카테고리별 핵심 특징과 타겟 고객을 CSV에 구조화해 입력
2. 전문적 톤으로 1차 생성 후, 감성적 톤으로 2차 생성해 비교
3. 채널별 미리보기에서 과장 표현, 부적절 표현을 제거하고 문안 확정
4. 확정 결과를 파일로 내보내 카테고리 온보딩 자료로 재사용

### 기대 결과
- **학습 비용 절감**: 신규 카테고리에서도 빠르게 실무 수준 초안 확보
- **품질 안정화**: 톤 전환과 재생성으로 카테고리 초기 문체를 빠르게 정착
- **재사용성 향상**: 내보낸 결과를 팀 내부 템플릿 자산으로 축적

---

## 추가 시나리오 C: 저성과 상품 문안 리프레시

### 상황
박수진 팀은 노출 대비 반응이 낮은 상품군 30개를 점검하고, 기존 문안을 리프레시해 클릭 유도를 개선하려고 합니다.

### 작업 흐름
1. 저성과 상품 목록을 CSV로 업로드
2. 기존 톤과 다른 톤앤매너(전문적 -> 감성적 등)로 재생성
3. CTA 문구와 길이를 조정해 채널별 버전을 확정
4. 복사/내보내기로 즉시 교체 배포

### 기대 결과
- **개선 반복 속도 향상**: 저성과 상품군을 짧은 주기로 재정비 가능
- **운영 부담 완화**: 전면 재작성 없이 핵심 문구 중심으로 빠르게 보정
- **실행력 강화**: 기획-수정-배포 사이클을 단축해 실험 빈도 증가

## 공통 기대사항

마케팅 팀장이 공통적으로 기대하는 가치:
1. **시간 절감**: 반복 문안 작업 시간을 70% 이상 단축
2. **품질 향상**: 채널별 특성에 맞는 전문적 표현 확보
3. **사용의 용이성**: 로그인 없이 빠르게 진입해 즉시 생성
4. **운영 효율**: 편집, 복사, 파일 내보내기까지 한 흐름에서 처리
5. **브랜드 일관성**: 팀 가이드 기준의 톤앤매너를 안정적으로 유지
