# Commerce AdPilot 사용자 시나리오

## 개요
이 문서는 Commerce AdPilot 사용자들의 실제 사용 사례와 기대하는 경험을 상세히 기술합니다. 각 시나리오는 사용자의 배경, 목표, 작업 흐름, 그리고 예상 결과를 포함합니다.

---

## 시나리오 1: 신규 온라인 판매자 - 김민준

### 사용자 프로필
- **이름**: 김민준
- **나이**: 28세
- **직업**: 제조업체 마케팅 담당자에서 온라인 판매로 전직
- **경험**: 온라인 판매 1개월차, 마케팅 경험은 있으나 콘텐츠 작성 경험 부족
- **목표**: 낮은 비용으로 전문적인 상품 설명과 광고 문구를 빠르게 생성

### 상황
김민준은 핸드메이드 가죽 제품을 온라인 쇼핑몰에 판매하기 시작했습니다. 첫 상품들이 올라가 있지만, 상품 설명이 단순하고 광고 문구가 매력적이지 않아 판매가 잘 안 되고 있습니다. 전문 디자이너나 카피라이터를 고용할 예산이 없어서 고민 중입니다.

### 작업 흐름

1. **상품 정보 입력**
   - Commerce AdPilot에 접속하여 회원가입 진행
   - 새로운 상품 추가 클릭
   - 상품명: "수제 카멜 가죽 지갑"
   - 카테고리: 패션/액세서리
   - 주요 특징 입력:
     - 100% 천연 가죽 사용
     - 손으로 만든 제품
     - 8개 카드 슬롯 포함
     - 독특한 디자인과 내구성
   - 타겟 고객: "20-40대 직장인, 프리미엄 제품 선호"

2. **콘텐츠 생성**
   - "콘텐츠 생성" 버튼 클릭
   - 플랫폼에서 제공하는 여러 콘텐츠 옵션으로부터 생성 요청:
     - 짧은 광고 문구 (50자)
     - 긴 상품 설명 (300자)
     - 소셜 미디어 포스트 (인스타그램, 페이스북용)
     - 해시태그 제안

3. **생성 결과 확인**
   - 약 5초 후 다양한 버전의 콘텐츠 수신
   - 예시:
     - **짧은 광고**: "매 순간이 특별한 순제 가죽 지갑. 한 땀 한 땀 정성이 담긴 프리미엄 액세서리"
     - **긴 설명**: "우리의 카멜 가죽 지갑은 최고 품질의 천연 가죽으로 정성스럽게 만들어집니다. 각 제품은 숙련된 장인의 손으로 개별 제작되며, 세련된 디자인과 뛰어난 내구성을 자랑합니다. 8개의 카드 슬롯으로 일상용품을 효율적으로 정리할 수 있으며, 시간이 지날수록 더욱 아름다운 색감으로 변해가는 천연 가죽의 매력을 느낄 수 있습니다."
     - **해시태그**: #수제가죽 #프리미엄지갑 #가죽제품 #핸드메이드 #천연가죽

4. **콘텐츠 편집**
   - 생성된 텍스트 중 마음에 들지 않는 부분 직접 수정
   - "독자적인 스타일" 추가로 입력
   - 수정된 콘텐츠 다시 생성 요청

5. **저장 및 활용**
   - 최종 확정된 콘텐츠 저장
   - 텍스트 파일로 내보내기 또는 클립보드에 복사
   - 온라인 쇼핑몰에 복사-붙여넣기로 적용

### 기대 결과
- **시간 절약**: 기존 30분 걸리던 작업을 2분으로 단축
- **품질 향상**: 전문적인 톤과 효과적인 메시지 구성
- **매출 증대**: 더 나은 설명으로 고객의 클릭율 및 구매 전환율 증가 기대
- **확신감**: "이 정도 콘텐츠라면 충분히 경쟁력 있겠다"는 생각으로 자신감 형성

## 시나리오 2: 마케팅 팀 리더 - 박수진

### 사용자 프로필
- **이름**: 박수진
- **나이**: 35세
- **직업**: 온라인 쇼핑몰 마케팅 팀 리더
- **경험**: 마케팅 경력 10년, 현재 팀의 5명이 담당할 상품 수 200개/월
- **목표**: 팀의 생산성 극대화, 콘텐츠 생성 시간 70% 단축

### 상황
박수진이 이끄는 마케팅 팀은 월 200개의 신상품 콘텐츠를 작성해야 합니다. 기존 방식으로는 인턴 1명과 경력사원 2명이 거의 전일제로 콘텐츠 작성에만 시간을 쏟고 있습니다. 이로 인해 더 중요한 전략적 작업(캠페인 기획, 성과 분석 등)에 집중하지 못하고 있는 상황입니다.

### 작업 흐름

1. **팀 계정 설정**
   - 기업용 플랜 가입 (팀 구성원 5명)
   - 팀 계정에서 브랜드 가이드라인 설정:
     - 톤: "친근하면서도 프로페셔널"
     - 주요 메시지: "고품질, 합리적 가격, 빠른 배송"
     - 금지 표현 및 필수 표현 입력

2. **대량 콘텐츠 생성 작업**
   - CSV 파일로 상품 정보 일괄 업로드 (200개)
   - 각 상품별로 다음 항목 포함:
     - 상품명, 카테고리, 주요 스펙, 가격대, 타겟층
   - "배치 생성" 모드 시작

3. **배치 생성 진행**
   - 시스템이 한 시간 동안 200개 상품의 콘텐츠 자동 생성
   - 각 상품마다 3가지 스타일 버전 제공:
     - 버전 A: 감정적 어필 (이야기 중심)
     - 버전 B: 이성적 어필 (스펙 중심)
     - 버전 C: 행동 촉구 (할인, 이벤트 강조)

4. **품질 검토 및 승인**
   - 팀원들이 AI 생성 콘텐츠를 검토
   - 각자 90개 상품 담당:
     - 톤 체크
     - 오타/문법 확인
     - 부문별 특성 반영 여부 확인
   - 필요시 AI로 재생성 또는 수동 수정

5. **A/B 테스트 준비**
   - 일부 상품의 여러 버전을 실제 판매 채널에 적용
   - A/B 테스트로 어느 스타일의 콘텐츠가 더 높은 전환율을 가져오는지 분석

6. **결과 통합**
   - 검증된 최고 성과 콘텐츠 버전을 메인으로 적용
   - 패턴 분석: "어떤 표현이 잘 팔리는가?"
   - 향후 AI 모델에 피드백 제공

### 기대 결과
- **시간 절감**: 기존 월 300시간 → 월 90시간 (70% 단축)
- **팀 재배치**: 절감된 시간을 전략 기획, 성과 분석 등 고부가가치 업무에 투입
- **일관성**: 전사 브랜드 가이드라인 준수한 일관된 톤의 콘텐츠
- **최적화**: 데이터 기반의 콘텐츠 스타일 최적화
- **ROI**: 생성 시간 단축으로 월간 수십만 원대의 인건비 절감

## 시나리오 3: 소상공인 - 이영미

### 사용자 프로필
- **이름**: 이영미
- **나이**: 42세
- **직업**: 온라인 화장품 판매 소상공인
- **경험**: 온라인 판매 3년, 기술에 약함, 수작업으로 모든 콘텐츠 작성 중
- **목표**: 남은 제품들의 효과적인 설명 작성, 최소한의 기술 학습

### 상황
이영미는 수입 화장품을 온라인에서 판매하고 있습니다. 아직 판매되지 않은 상품들이 100개 정도 남아 있는데, 일일이 손으로 설명을 작성하기에는 시간이 너무 많이 걸립니다. 특히 각 상품의 성분, 효능, 사용 방법에 대한 전문적인 설명을 작성하는 것이 어렵습니다.

### 작업 흐름

1. **간단한 회원가입**
   - 이메일과 비밀번호로 계정 생성 (복잡한 인증 없음)
   - 기본 언어 설정: 한국어

2. **상품 정보 간단히 입력**
   - 상품명: "비타민C 고함유 에센스"
   - 브랜드: "ABC스킨케어"
   - 주요 성분: "비타민C, 히알루론산"
   - "사용 대상" 선택: 복합성 피부, 민감성 피부
   - "상품 특징" 선택 (체크박스):
     - ✓ 항산화 효과
     - ✓ 보습 효과
     - ✓ 빠른 흡수

3. **자동 콘텐츠 생성**
   - "생성하기" 버튼 클릭
   - 약 3초 후 설명, 사용법, 추천 대상 등 생성됨

4. **간단한 미세 조정**
   - 생성된 텍스트 읽어보기
   - 원하지 않는 부분 체크 후 "다시 생성" 클릭
   - 취향에 맞는 버전 최종 선택

5. **일괄 적용**
   - "복사" 버튼으로 클립보드에 복사
   - 쇼핑몰 관리 페이지에 붙여넣기

### 기대 결과
- **시간 효율**: 상품당 기존 15분 → 1분 (94% 단축)
- **100개 상품 처리**: 총 25시간 → 1.5시간으로 단축
- **품질 향상**: 직접 작성하는 것보다 더 전문적이고 체계적인 설명
- **빠른 판매**: 남은 재고를 빠르게 온라인에 올려 판매 시작 가능
- **스트레스 감소**: 기술 활용에 대한 부담이 적어 부담 없이 사용

## 시나리오 4: 마케팅 에이전시 - 최동욱

### 사용자 프로필
- **이름**: 최동욱
- **나이**: 30세
- **직업**: 마케팅 에이전시 CD (Creative Director)
- **경험**: 에이전시 경력 7년, 다양한 클라이언트 경험
- **목표**: 클라이언트 프로젝트의 빠른 초안 생성, 창의적 아이디어 발전

### 상황
최동욱은 소매 클라이언트들을 위해 다양한 마케팅 캠페인을 진행합니다. 각 클라이언트마다 100-500개의 상품 콘텐츠가 필요한데, 전체 프로세스에서 초안 생성 단계가 가장 오래 걸립니다. 더 많은 시간을 창의적 전략 개발과 개선에 활용하고 싶습니다.

### 작업 흐름

1. **프로젝트 및 클라이언트 설정**
   - 새 프로젝트 생성: "OO쇼핑몰 신년 캠페인"
   - 클라이언트 특성 입력:
     - 톤: "젊고 트렌디함"
     - 타겟층: "20-30대 패션 관심층"
     - 캠페인 테마: "신년 신상품 주간"

2. **대량 상품 정보 생성**
   - 200개 상품 정보 엑셀 파일 업로드
   - 각 상품: 이름, 카테고리, 가격, 주요 특징

3. **다양한 스타일 옵션으로 콘텐츠 생성**
   - 스타일 1: "감성적/스토리텔링"
   - 스타일 2: "활동적/행동 촉구"
   - 스타일 3: "신뢰성/품질 강조"
   - 각 스타일별로 200개 상품 전체 콘텐츠 자동 생성

4. **창의적 개선 단계**
   - AI 생성 콘텐츠를 기반으로 필요한 부분만 수정/개선
   - 예: 일부 상품의 감정적 표현을 더 강하게 조정
   - 캠페인 코드/프로모션 정보 추가
   - 특별 이벤트 메시지 삽입

5. **클라이언트 프리젠테이션**
   - 3가지 콘셉트 버전 준비
   - 각 버전의 예시 20개 상품 콘텐츠 시각화
   - "AI로 빠르게 생성하되, 전략적으로 최적화된 콘텐츠"로 설명

6. **피드백 반영 및 최종 작업**
   - 클라이언트 선택 콘셉트에 맞춰 200개 전체 최종 조정
   - 최종 파일 내보내기 및 전달

### 기대 결과
- **프로젝트 기간 단축**: 기존 2주 → 3일 (85% 단축)
- **비용 절감**: 적은 인력으로 더 많은 프로젝트 수행 가능
- **창의성 집중**: 초안 생성에서 개선/최적화로 에너지 재배치
- **클라이언트 만족**: 빠르고 효율적인 서비스 제공으로 만족도 증가
- **반복 작업 최소화**: 여러 버전 빠르게 생성 → 클라이언트 선택지 제공

## 공통 기대사항

모든 사용자가 공통으로 기대하는 가치:
1. **시간 절감**: 콘텐츠 작성 시간 70% 이상 단축
2. **품질 향상**: AI의 전문적이고 체계적인 표현 활용
3. **사용의 용이성**: 복잡한 기술 없이 직관적 사용 가능
4. **비용 효율**: 전문가 고용 대비 매우 저렴한 비용
5. **일관성**: 브랜드 가이드라인에 맞춘 일관된 톤의 콘텐츠
