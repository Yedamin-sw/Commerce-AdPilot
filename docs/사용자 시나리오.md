# BioVerse 사용자 시나리오

## 개요
이 문서는 BioVerse의 핵심 타깃인 제약 회사 마케팅 팀장을 중심으로, 실제 업무 상황에서의 사용 시나리오와 기대 성과를 정리합니다. 모든 시나리오는 MVP 범위(로그인 없는 진입, CSV 업로드, 멀티 채널 동시 생성, 편집/복사/내보내기) 안에서 작성됩니다.

---

## 대표 시나리오: 신약/신제품 멀티채널 마케팅 동시 운영

### 사용자 프로필
- **이름**: 김의준
- **나이**: 38세
- **직업**: 제약 회사 마케팅 팀 리더
- **경험**: 의약 마케팅 경력 12년, 현재 팀의 6명이 담당할 신약/신제품 연간 20개
- **목표**: 규제 기준에 맞는 정확한 콘텐츠를 채널별로 동시 생성해 마케팅 콘텐츠 작성 시간을 70% 이상 단축

### 상황
김의준 팀은 신약 출시, 적응증 확대, 신제품 런칭 시마다 의사·약사·환자 등 이해관계자별 맞춤형 콘텐츠를 블로그, 의료전문가 커뮤니티, SNS(인스타그램, 페이스북)에 배포해야 합니다. 기존 방식은 채널별로 규제 준수 사항을 반영해 문안을 따로 작성해 반복 작업이 크고, 컴플라이언스 검수까지 포함하면 런칭까지의 리드타임이 길어집니다. 팀은 초안 작성 시간을 최소화하고 규제 기준 내 신뢰도 높은 콘텐츠 생성에 집중하려고 합니다.

### 작업 흐름

1. **즉시 진입 및 작업 시작**
   - 대표 페이지 접속 후 로그인 없이 바로 생성 화면으로 이동
   - 팀의 기본 작성 기준 입력:
     - 기본 톤: "신뢰도 높고 과학적 근거 기반의 전문적 톤"
     - 핵심 메시지: "임상 효과, 안전성, 환자 삶의 질 개선"
     - 컴플라이언스 가이드: 금지 표현(과장 표현), 필수 표현(부작용 공시), 의약품 법규 준수 기준

2. **의약품/제품 정보 입력(대량 업로드)**
   - CSV 파일로 신약/신제품 정보 업로드
   - 필수 항목: 약품명, 적응증, 임상 데이터 요약, 부작용/금기사항, 타겟 청중(의사/약사/환자)
   - 핵심 신약은 개별 입력으로 규제 사항, 가이던스 기준, 강조 포인트 추가

3. **멀티 채널 콘텐츠 동시 생성**
   - 생성 채널 선택: 의료 커뮤니티 블로그, 인스타그램(환자 교육), 페이스북(일반 대중)
   - 톤앤매너 선택: 의료전문가용(학술적)/환자용(친근한)/일반대중용(교육적)
   - 생성 버튼 클릭 후 약품별 채널 맞춤형 콘텐츠 동시 생성
   - 초기 결과는 약품당 10초 이내 확인

4. **결과 검토 및 컴플라이언스 검증**
   - 생성된 콘텐츠를 채널별 미리보기로 확인
   - 의약품 광고 법규 준수, 과장/허위 표현 제거, 부작용 공시 여부 확인
   - 컴플라이언스 팀 가이드에 맞게 편집 및 필요한 항목만 재생성

5. **배포 준비 및 활용**
   - 확정 콘텐츠를 클립보드 복사 또는 텍스트 파일로 내보내기
   - 각 채널 운영 도구에 붙여넣어 즉시 게시 준비
   - 입력부터 최종 복사/내보내기까지 약품별 작업을 10분 내 완료

### 기대 결과
- **시간 절감**: 채널별 개별 작성 대비 신약 마케팅 콘텐츠 작업 시간 70% 이상 단축
- **처리량 향상**: 연간 20개 신약/신제품을 동일 인력으로 안정적으로 처리
- **컴플라이언스 강화**: 채널별 톤 차이는 유지하면서 의약품 광고 법규를 일관되게 준수
- **실행 속도 개선**: 초안 작성보다 규제 검수/개선 중심으로 업무 전환
- **운영 효율 증대**: 복사/내보내기 중심의 단순한 배포 흐름으로 출시 리드타임 단축

## 추가 시나리오 A: 규제 이슈 긴급 대응

### 상황
식약청 공시가 나갔고, 약품 안전성 정보 업데이트가 2일 내 완료되어야 합니다. 김의준 팀은 급하게 기존 5개 약품의 채널별 콘텐츠를 규제 요구사항에 맞게 수정해야 합니다.

### 작업 흐름
1. 기존 CSV를 불러와 영향받는 약품만 추려 재업로드
2. 톤앤매너를 "안전성 중심" 으로 선택하고 채널별 콘텐츠 동시 생성
3. 새로운 부작용 정보, 사용 주의사항, 환자 안내 문구를 편집 단계에서 일괄 반영
4. 최종 콘텐츠를 클립보드 복사/파일 내보내기로 운영툴에 즉시 반영

### 기대 결과
- **규제 준수**: 규제 변경 사항 반영 기한 전 전 채널 콘텐츠 업데이트 완료
- **속도 확보**: 긴급 작업에서도 입력-내보내기 리드타임을 10분 내 수준으로 유지
- **신뢰도 유지**: 안전성 정보를 모든 채널에 일관되게 적용

## 추가 시나리오 B: 신약 품목별 맞춤형 교육 콘텐츠

### 상황
회사에 새로운 적응증 확대 약품(예: 당뇨병 합병증 치료제) 10개가 추가되었습니다. 기존 임상 프레임이 없어 의사용, 약사용, 환자용 3개 채널별 교육 콘텐츠 초안 작성 부담이 큽니다.

### 작업 흐름
1. 신약 10개의 임상 데이터, 적응증, 부작용을 CSV에 구조화해 입력
2. 의료전문가용 톤으로 1차 생성 후, 환자용 톤으로 2차 생성해 비교
3. 채널별 미리보기에서 과장/허위 표현, 규제 위반 문구를 제거하고 문안 확정
4. 확정 결과를 파일로 내보내 신약 온보딩 자료 및 내부 표준 템플릿으로 재사용

### 기대 결과
- **학습 비용 절감**: 신약 적응증에서도 빠르게 규제 준수 수준의 콘텐츠 초안 확보
- **품질 안정화**: 톤 전환과 재생성으로 약품별 메시지 일관성을 빠르게 정착
- **재사용성 향상**: 내보낸 결과를 팀 내부 템플릿 자산으로 축적해 유사 약품 마케팅 시 활용

## 추가 시나리오 C: 환자 검색 이슈 대응 콘텐츠 리프레시

### 상황
김의준 팀은 검색 트렌드와 환자 커뮤니티 피드백을 바탕으로 저참여율 약품 8개의 환자 교육 콘텐츠를 리프레시하려고 합니다.

### 작업 흐름
1. 저참여 약품 목록과 새로운 환자 관심사 키워드를 CSV로 업로드
2. 기존 톤과 다른 톤앤매너(학술적 -> 감정 공감형으로 전환 등)로 재생성
3. 환자 입장의 공감 메시지, CTA 문구와 길이를 조정해 채널별 버전 확정
4. 복사/내보내기로 즉시 교체 배포

### 기대 결과
- **개선 반복 속도 향상**: 저참여 콘텐츠를 짧은 주기로 재정비 가능
- **운영 부담 완화**: 전면 재작성 없이 메시지 리프레임 중심으로 빠르게 개선
- **실행력 강화**: 트렌드 대응-콘텐츠 수정-배포 사이클을 단축해 환자 접근성 개선

## 공통 기대사항

제약 마케팅 팀장이 공통적으로 기대하는 가치:
1. **시간 절감**: 반복 콘텐츠 작성 시간을 70% 이상 단축
2. **컴플라이언스 강화**: 채널별 특성에 맞으면서 의약품 법규를 준수하는 안전한 표현 확보
3. **사용의 용이성**: 로그인 없이 빠르게 진입해 즉시 생성
4. **운영 효율**: 편집, 복사, 파일 내보내기까지 한 흐름에서 처리
5. **메시지 일관성**: 팀 가이드 및 규제 기준에 따른 톤앤매너를 안정적으로 유지
