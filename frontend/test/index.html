기본

GET / 서버 헬스체크

인증 (Auth)

POST /api/auth/signup 회원가입
POST /api/auth/login 로그인 → 토큰 자동 저장
POST /api/auth/logout 로그아웃 (저장된 토큰 사용)
POST /api/auth/deleteUser 회원 탈퇴

광고 생성 (Generate)

POST /api/generate/normal 단일 광고 카피 생성
POST /api/generate/platform 선택한 플랫폼 모두에 대해 병렬 생성
POST /api/generate/csv CSV / Excel 파일 일괄 생성
GET /api/generate/csv/status/{job_id} 비동기 작업 상태 조회
GET /api/generate/csv/preview/{job_id} 완료된 항목 미리보기 (페이징)
POST /api/generate/csv/retry/{job_id} 실패 항목 재시도

히스토리 (History)

POST /api/history/save 생성된 광고 콘텐츠 저장
GET /api/history/list/product 상품 목록 조회 (페이지)
GET /api/history/list/content/{product_id} 상품별 광고 콘텐츠 조회
